## Zweck des Projekts

Das Projekt nutzt das Model Context Protocol (MCP), um Claude Desktop mit Blender zu verbinden und Aufgaben in 3D-Szenen zu automatisieren. Dafür wird eine bestehende Schnittstelle erweitert und eine praxisorientierte Anleitung zur Einrichtung und Nutzung erstellt.

---

## Features

- Anwendungsbeispiel in Form einer visuellen digitalen Vorstellung
- Anwendungsbezogene Übungsangabe für Schüler:innen
